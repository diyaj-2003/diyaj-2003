`01` &nbsp;&nbsp; **Optimising Hippocampal Stimulation for Memory Consolidation**

<sub>MEng Individual Research Project ‚Äî May 2025 &nbsp;¬∑&nbsp; Supervisor: Dr. Ines Violante</sub>

`02` &nbsp;&nbsp; **Virtual Pacemapping ‚Äî ECG Alignment for Arrhythmic Source Localisation**

<sub>BEng Thesis ‚Äî April 2024 &nbsp;¬∑&nbsp; Supervisors: Dr. Martin Bishop, Fernando Campos, Shuang Qian</sub>
