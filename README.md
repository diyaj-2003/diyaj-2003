<sub>ARCHIVED RESEARCH</sub>

`01` &nbsp; **Optimising Hippocampal Stimulation for Memory Consolidation**
<sub>MEng — May 2025 &nbsp;·&nbsp; Supervisor: Dr. Ines Violante</sub>

Personalised modelling of temporal interference electric fields using subject-specific MRI/DWI data across 26 participants. Memory improvement correlated with hippocampal field strength in low encoders (r = 0.64, p = 0.026).

`02` &nbsp; **Virtual Pacemapping — ECG Alignment for Arrhythmic Source Localisation**
<sub>BEng — April 2024 &nbsp;·&nbsp; Supervisors: Dr. Martin Bishop, Fernando Campos, Shuang Qian</sub>

Aligned simulated and clinical 12-lead ECGs across 1,000 cardiac mesh nodes to localise ventricular tachycardia sources non-invasively. QRS start alignment + Pearson/L2/E12 outperformed DTW with shifting.
